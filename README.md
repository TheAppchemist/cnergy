Android Cnergy v0.8.2
=====================

A powerful HTTP resource loading library for Android

Download
--------

Grab via Gradle:
```groovy
compile 'com.appchemy.cnergy:app:0.8.2'
```
or Maven:
```xml
<dependency>
  <groupId>com.appchemy.cnergy</groupId>
  <artifactId>cnergy</artifactId>
  <version>0.8.2</version>
</dependency>
```
